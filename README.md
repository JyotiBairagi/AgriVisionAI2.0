# 🌱 AgriVision AI

AgriVision AI is an AI-powered smart farming assistant that helps farmers analyze crop growth stages and receive intelligent farming recommendations. The application combines image analysis with Generative AI to provide irrigation advice, fertilizer suggestions, disease prevention tips, and best farming practices.

---

## 🚀 Features

- 🌿 Crop Growth Stage Analysis
- 📷 Crop Image Upload
- 🤖 AI-powered Farming Recommendations using Ollama (Gemma 3:4B)
- 💧 Irrigation Suggestions
- 🌱 Fertilizer Recommendations
- 🦠 Disease Prevention Tips
- 📜 Analysis History
- 🌐 Multi-language AI Support
- 💻 Responsive User Interface

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/JyotiBairagi/AgriVisionAI2.0.git
```

---

### 2. Open Project

Open the project folder in VS Code.

---

### 3. Install Ollama

Download and install Ollama

https://ollama.com/

---

### 4. Pull Gemma Model

```bash
ollama pull gemma3:4b
```

---

### 5. Start Ollama

```bash
ollama run gemma3:4b
```

---

### 6. Run Frontend

Simply open

```
index.html
```

or use Live Server in VS Code.

## 📷 How It Works

1. Login/Register
2. Upload Crop Image
3. Confirm Crop Image
4. Analyze Crop
5. Detect Crop Growth Stage
6. Generate AI Recommendation
7. View Irrigation & Fertilizer Advice
8. Save Analysis History
